# SoundCloudMusicBot 
A simple telegram bot made using telegraf.js and soundcloud-scraper. The bot allows you to search and download music from soundcloud.

###### git clone https://github.com/AntonPodlatov/SoundCloudMusicBot.git
###### npm install
create a .env file in bor_dir and put the token on it

create "songs" directory in bot_dir

node bot - start bot
